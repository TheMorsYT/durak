# 🃏 Durak Card Game

## 📄 About

This project is a digital adaptation of the popular card game **"Durak"** (Fool). The main goal of this project is to implement standard card game logic, create a highly functional user interface, and develop a competitive artificial intelligence using the **Unity Engine**.

## ✨ Current Features

* **Cross-Platform Support:** Fully playable on both **Windows (x64)** and **Android**.
* **Discord Integration:** Show your current game status to your friends using official Discord Rich Presence.
* **Player Profiles & Stats:** Customize your identity with Nicknames and Avatars, and track your overall match statistics.
* **Dynamic Timers:** Visual turn timers ensure a structured and paced gameplay experience for both attackers and defenders.
* **Smart Hand Sorting:** Choose between **Manual**, **By Rank**, or **By Suit** sorting methods via the settings menu.
* **Game Modes:** Full implementation of both "Throw-in" (Podkidnoy) and **"Transferable" (Perevodnoy)** rules.
* **Deck Options:** Support for standard 36-card decks and fast-paced **24-card decks**.
* **Advanced AI Opponent:** Play against a computer opponent that handles transfers, manages trump cards strategically, and features realistic "thinking" delays.
* **Settings & Localization:** Full UI support for English and Ukrainian languages.
* **Improved UI:** Responsive layouts and enlarged buttons optimized for mobile touchscreens.

## 🎮 Planned Features

* **Multiplayer Mode:** Planning **Local/LAN (Peer-to-Peer)** and online multiplayer for playing with friends without external servers.
* **Linux Port:** Exploring the possibility of a native build for Linux-based systems.
* **Visual Polish:** Adding more card animations and particle effects.
* **Further AI & Logic Polish:** Continuing to refine bot difficulty and edge-case game rules based on the new core architecture.

## 🛠️ Technologies Used

* **Engine:** Unity 6 (6000.3.10f1) *(Note: Versions prior to v5.0-alpha used Unity 2019.2.8f1)*
* **Language:** C#
* **Integrations:** Discord Game SDK

## 🚀 How to Install & Play

**For Windows:**
1.  **Download:** Go to the [Releases](#) page and download the `Durak_v.zip`.
2.  **Unpack:** Extract the archive into any folder.
3.  **Run:** Launch the game via `Durak.exe`.

**For Android:**
1.  **Download:** Grab the `.apk` file from the latest Release.
2.  **Install:** Open the file on your smartphone and allow installation from unknown sources.

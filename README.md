## üìÑ About
This project is a digital adaptation of the popular card game **"Durak"** (Fool).
The main goal of this project is to implement standard card game logic, create a highly functional user interface, and develop a competitive artificial intelligence using the **Unity Engine**.

## ‚ú® Current Features
* **Cross-Platform Support:** Fully playable on both **Windows (x64)** and **Android**.
* **Smart Hand Sorting:** Choose between **Manual**, **By Rank**, or **By Suit** sorting methods via the settings menu.
* **Game Modes:** Full implementation of both "Throw-in" (Podkidnoy) and **"Transferable" (Perevodnoy)** rules.
* **Deck Options:** Support for standard 36-card decks and fast-paced **24-card decks**.
* **Advanced AI Opponent:** Play against a computer opponent that handles transfers and manages trump cards strategically.
* **Settings & Localization:** Full UI support for English and Ukrainian languages.
* **Improved UI:** Responsive layouts and enlarged buttons optimized for mobile touchscreens.

## üéÆ Planned Features
* **Multiplayer Mode:** Planning **Local/LAN (Peer-to-Peer)** multiplayer for playing with friends on the same network without external servers.
* **Linux Port:** Exploring the possibility of a native build for Linux-based systems.
* **Visual Polish:** Adding more card animations and particle effects.
* **Game Improvements:** General logic optimization and AI difficulty tweaks.
